# Rise — односторінковий сайт (Landing)
Це готовий шаблон односторінкового сайту для особистого бренду **Rise** (прикраси).

## Що в архіві
- `index.html` — головна сторінка (українською)
- `styles.css` — стилі
- `script.js` — простий JS (модальне вікно, обробка форми)
- `README.md` — інструкції
- `rise_site.zip` — архів для завантаження

## Як розмістити на GitHub Pages
1. Створи репозиторій на GitHub (наприклад `rise-site`).  
2. Завантаж файли з цього архіву в корень репозиторію (натисни Upload files або скористайся git).  
3. На GitHub зайди в `Settings` → `Pages` (або `Deploy` → `Pages`) і обери гілку `main`/`master` і папку `/ (root)`.  
4. Після декількох хвилин сайт буде доступний за `<your-username>.github.io/<repo>`.

## Де додати твої зображення та логотип
Цей шаблон використовує текстовий логотип `Rise`. Якщо хочеш додати свій логотип або фото прикрас, помістіть їх у папку `assets/` та вставте теги `<img src="assets/your-photo.jpg" alt="...">` у картки товарів або в хедер.

## Налаштування кольорів
Основні значення кольорів знаходяться в `:root` (styles.css). Можеш змінювати `--accent`, `--fg` тощо.
